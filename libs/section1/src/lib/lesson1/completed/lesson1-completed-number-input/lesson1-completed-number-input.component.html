The html we added will limit users when using the up and down arrow keys, but a user will still be able to type in numbers that greater than our max, or lower than our min, or are not multiples of 5.

Let's fix this by subscribing to the `valueChanges` of our FormControl and adjusting the value held by the control when it does not confirm to the limitations that we want. To make things manageable, let's also set the `updateOn` config option of our FormControl to 'blur'.
