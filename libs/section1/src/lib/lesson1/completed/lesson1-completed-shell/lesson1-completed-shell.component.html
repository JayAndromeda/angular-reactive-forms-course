Before moving on, let's take a moment to checkout the `DefaultValueAccessor`.

This is the implementation of the `ControlValueAccessor` interface that the Angular framework is using in the text input example. For bonus work, go modify some of the Angular source code so you can see how and when some of these methods are being called.

Looking at the Angular.io documentation for `DefaultValueAccessor`, we can see via the "Selectors" section that this is the `ControlValueAccessor` used for non-checkbox input elements and textarea elements. In our next example, let's try creating a number input:
